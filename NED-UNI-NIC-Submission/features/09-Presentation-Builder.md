# Feature Document: Presentation Builder

## Overview

The Presentation Builder is ShadowTalk's **AI-powered slide deck generator** that produces McKinsey-tier presentations with bespoke HTML/CSS for every slide. It follows a rigorous 4-phase workflow to ensure professional-grade output.

## 4-Phase Workflow

```
Phase 1: RESEARCH     → Gather data, statistics, market insights
Phase 2: NARRATIVE    → Structure 10-slide narrative arc
Phase 3: CODE         → Generate unique HTML/CSS per slide
Phase 4: POLISH       → Refine design, add transitions, export
```

## 10-Slide Narrative Arc

| Slide # | Purpose | Content |
|---------|---------|---------|
| 1 | Title | Company name, tagline, visual identity |
| 2 | Problem | Market pain point with statistics |
| 3 | Solution | Product overview and value proposition |
| 4 | Deep Dive | Technical architecture or process |
| 5 | Comparison | Competitive landscape analysis |
| 6 | Opportunity | Market size and growth potential |
| 7 | Metrics | Key performance indicators |
| 8 | Business Model | Revenue streams and unit economics |
| 9 | Roadmap | Development timeline and milestones |
| 10 | CTA | Call to action and next steps |

## Key Capabilities

| Capability | Description |
|-----------|-------------|
| **Bespoke HTML/CSS** | Every slide is unique, custom-designed code |
| **SVG Visualizations** | Data charts, diagrams, and icons as inline SVG |
| **Gemini 2.5 Pro** | Powered by Google's most capable AI model |
| **PPTX Export** | Professional PowerPoint export via pptxgenjs |
| **Content Density** | McKinsey-level metrics, citations, and analysis |
| **JSON Sanitization** | Specialized regex layer for clean AI output |
