# Feature Document: AI Image Generation

## Overview

ShadowTalk includes **AI-powered image generation** directly in the chat interface. Users can create images from text descriptions — more generous free-tier allowance than any competitor.

## Key Capabilities

| Capability | Description |
|-----------|-------------|
| **Text-to-Image** | Generate images from natural language descriptions |
| **In-Chat Integration** | Images generated directly in conversation flow |
| **Multiple Styles** | Photorealistic, illustration, abstract, etc. |
| **Download** | One-click download of generated images |
